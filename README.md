> A small interactive apology page built with browser-native HTML, CSS, and JavaScript.

## Overview

Sorry is a compact frontend project focused on simple emotion, interaction, and UI effects. It is a good mini-project for practicing DOM behavior, styling, and lightweight browser interactions.

## Highlights

- Small, readable frontend project
- Interactive browser-based UI
- Good for HTML/CSS/JavaScript practice
- Simple project scope for beginners
- Easy to deploy as a static site

## Built For

- Frontend fundamentals
- Creative mini-projects
- Static web page practice
- Portfolio warm-up projects

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser DOM APIs

## Quick Start

```powershell
git clone https://github.com/akyourowngames/Sorry.git
cd Sorry
```

## Run Locally

```powershell
Open index.html in your browser
Or serve locally with: npx serve .
```

## Project Structure

```text
index.html  page markup
style.css   styling
script.js   browser interaction logic
assets/     optional images and resources
```
